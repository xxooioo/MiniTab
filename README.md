# MiniTab - Chrome 新标签页扩展

> 简洁、优雅、高效的新标签页扩展

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-brightgreen)](https://chrome.google.com/webstore)

## ✨ 核心特性

- **多标签页管理** - 创建多个标签页分类管理，支持拖拽排序和滚轮切换
- **快捷方式** - 智能图标获取，支持拖拽排序、分组管理、右键快捷操作
- **文字图标** - 1-3 个字符，15 种颜色，自动生成精美图标
- **智能搜索** - 多引擎支持（Google/Bing/百度/自定义）
- **个性化** - 30 种渐变背景或自定义图片，透明度和布局可调
- **数据管理** - 备份恢复、书签导入、本地存储
- **快速添加** - 点击扩展图标一键添加当前网站
- **中键打开** - 支持鼠标滚轮点击打开快捷方式

## 🚀 安装

### 从源码安装

1. 克隆或下载项目
   ```bash
   git clone https://github.com/xxooioo/MiniTab.git
   ```

2. 打开 Chrome 扩展管理页面 `chrome://extensions/`，开启"开发者模式"

3. 点击"加载已解压的扩展程序"，选择 MiniTab 文件夹

4. 打开新标签页即可使用

## 📖 使用指南

### 标签页管理

- **切换** - 点击标签页名称或滚轮切换（需滚动到页面底部）
- **排序** - 拖拽标签页调整顺序
- **新建** - 点击底部 `+` 按钮
- **编辑/删除** - 右键点击标签页

### 快捷方式

#### 添加方式
1. **扩展图标**（推荐）- 访问网站，点击工具栏扩展图标，自动填充信息
2. **手动添加** - 点击快捷方式区域右上角 `+` 按钮

#### 图标选择
- **自动获取** - 留空自动获取高质量图标
- **输入 URL** - 使用自定义图标地址
- **上传图片** - 选择本地图片（最大 2MB）
- **文字图标** - 点击 **T** 按钮，输入 1-3 个字符，选择颜色

#### 分组管理
- **创建分组** - 拖拽一个图标到另一个图标上
- **添加到分组** - 拖拽图标到分组上
- **打开分组** - 点击分组图标
- **分组内排序** - 在分组内拖拽图标
- **移出分组** - 拖拽图标至分组外后松开或通过右键菜单操作
- **移出/移动** - 右键点击图标选择操作

### 书签导入

1. 打开设置 → 数据备份与恢复 → 导入浏览器书签
2. 自动将书签文件夹转换为分组
3. 追加到当前标签页最后

> 💡 导入前建议先备份现有数据

### 搜索

- **关键词** - 直接输入，按回车使用默认搜索引擎
- **网址** - 输入完整 URL，按回车直接访问
- **快捷键** - `Cmd/Ctrl + K` 聚焦搜索框

### 个性化设置

点击右下角 ⚙️ 打开设置：

- **背景** - 上传图片、随机渐变、恢复默认
- **搜索引擎** - Google / Bing / 百度 / 自定义
- **搜索框透明度** - 0-50% 可调
- **图标区域宽度** - 每行 6-12 个图标
- **自动隐藏** - 侧边栏和设置按钮
- **数据管理** - 备份/恢复（支持覆盖/合并）、书签导入

## ⌨️ 快捷键

- `Cmd/Ctrl + K` - 聚焦搜索框
- `Cmd/Ctrl + T` - 新标签页
- `ESC` - 关闭弹窗

## 🔧 技术栈

- 原生 JavaScript（无框架）
- CSS3（Flexbox + Grid）
- Chrome Extension Manifest V3
- Chrome Storage API

## 🔧 故障排查

### 图标加载失败
- 自动尝试多种方式获取
- 可手动输入 URL 或上传图片
- 使用文字图标功能

### 数据丢失
- 定期备份数据
- 卸载扩展会清除数据

### 拖拽无效
- 确保页面完全加载
- 刷新页面重试
- 检查浏览器控制台

### 书签导入失败
- 确保扩展有 bookmarks 权限
- 刷新页面重新导入
